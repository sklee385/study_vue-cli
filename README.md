# study_vue-cli

## install 

```bash
npm install -g @vue/cli
# version check 
vue --version
```

## prototyping 
> 프로토 타입핑에만 사용 권장

- create file is App.vue
    ```html
    <template>
    <h1>Hello!</h1>
    </template>
    ```
- server start 
    ```bash
    vue serve
    ```

- file name is not App.vue. server start 
    ```bash
    vue serve [filename].vue
    ```

- vue build 
    ```bash
    vue build [filename].vue
    ```

## creating project 
    ```bash 
    vue create [projectname]
    ```

## use GUI 
    > vue ui command use.   
    > manage projects with a graphical interface.
    ```bash
    vue ui
    ```
